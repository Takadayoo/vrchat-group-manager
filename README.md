# VRChat Group Manager

VRChatのグループ一括管理するデスクトップアプリケーション（β版）

## 機能

- VRChat APIトークンでログイン
- 所属グループの一覧表示
- グループメンバーの可視状態を一括変更
  - 公開 / フレンドのみ / 非公開

## 対象ユーザー

- 複数のVRChatグループを管理している方
- グループの公開設定をまとめて変更したい方

## 対応環境

- OS: Windows 10 / 11（64bit）
- VRChat: PC版のみ対応

※ macOS 用の設定は含まれていますが、動作確認は行っていません。

## セキュリティについて

- VRChat APIトークンは **Windows 資格情報マネージャー** に保存されます
- アプリ内・設定ファイル・平文ファイルには保存されません
- アンインストール時に、資格情報を削除するか選択できます

※ トークンの取り扱いには十分注意してください。第三者に共有しないでください。

## 作成されるフォルダ

本アプリは以下のフォルダを作成します。

```
AppData\Local\VRChat Group Manager
```

- アプリの動作に必要なデータを保存するために使用されます

## インストール・アンインストール

- NSIS インストーラーを使用しています
- アンインストール時に以下を選択できます
  - アプリフォルダの削除
  - Windows 資格情報マネージャーに保存されたトークンの削除

## 技術スタック

- **フロントエンド**: React 19 + TypeScript
- **バックエンド**: Rust (Tauri 2.0)
- **UI**: Tailwind CSS
- **ビルドツール**: Vite

## 開発環境のセットアップ

### 必要な環境

- Node.js (v18以上)
- Rust (最新安定版)
- Tauri CLI

### インストール

```bash
# 依存関係のインストール
npm install

# Tauri CLIのインストール（グローバル）
cargo install tauri-cli --version "^2.0.0"
```

### 開発サーバーの起動

```bash
cargo tauri dev
```

### ビルド

```bash
cargo tauri build
```

## 使い方

1. VRChatにブラウザでログイン
2. VRChatのauthページから`token`トークンを取得
3. アプリにトークンを入力してログイン
4. グループを選択して可視状態を一括変更

## 注意事項

- 本アプリは **非公式ツール** です
- β版のため、予期しない不具合が発生する可能性があります
- VRChatの仕様変更により、突然動作しなくなる場合があります
- 利用は自己責任でお願いします
